# EcoTrack: Waste & Recycling Management System

A full-stack web application designed to streamline municipal waste collection, improve recycling transparency, and provide a centralized platform for resident engagement.

## 🚀 Project Overview
The **Waste Collector & Recycling Management System** bridges the gap between residents and waste management services. It replaces manual reporting with a digital interface where residents can track schedules, submit feedback, and manage recycling categories in real-time.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, Jinja2 Templating
* **Database:** SQLAlchemy / SQLite
* **Design Patterns:** Object-Oriented Programming (OOP), MVC Architecture

## ✨ Key Features
* **Resident Feedback System:** Dedicated portal for residents to submit complaints or feedback regarding collection services.
* **Live Complaint Tracking:** Backend logic to handle, categorize, and track the status of reported issues.
* **Collection Scheduling:** Automated management of waste collection dates and recycling categories.
* **Scalable Database Models:** Optimized schema for managing users, collectors, and recycling logs.
